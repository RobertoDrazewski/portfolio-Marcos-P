## 🛠️ Key Features
* **Professional Dark Mode:** High-end UI designed with a midnight palette for a modern tech feel.
* **Real-time Availability Status:** Animated "Online" pulse indicator on profile picture.
* **Interactive Contact Hooks:** * **Hire Me:** Smart logic that triggers the default mail client and copies the email to the clipboard as a fallback.
    * **WhatsApp Integration:** Direct link to chat for instant networking.
* **Responsive Tech Stack:** Comprehensive list of 15+ enterprise tools including Microsoft 365, Azure, Entra ID, and Citrix.
* **Hybrid Profile Section:** Highlighting the unique blend of IT precision and creative drive (Studio Owner/Drummer).

## 🧰 Tech Stack (IT)
* **Administration:** Microsoft 365, Azure, Entra ID, Intune.
* **Infrastructure:** Active Directory, GPO, Citrix VDI, Advanced Networking.
* **ITSM:** ServiceNow, Halo ITSM, NinjaOne.
* **Automation:** PowerShell.
